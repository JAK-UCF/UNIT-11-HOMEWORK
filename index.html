Summary: Latitude vs. X

The purpose of Latitude was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, data was pulled from the OpenWeatherMap API to assemble a dataset of more than 650 cities.

After assembling the dataset, various aspects of weather by latitude were plotted using Matplotib. Plotted evaluations were completed for temperature, humidity, cloudiness, and wind speed.

This site provides the visualizations created during the project with commentaries summarizing each analysis plot. The data used in plotting can be reviewed by visiting the Data page.
